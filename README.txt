Pentru a rula aplicatia sunt necesare: un server local cu suport pentru php si un server de oracle database 11g (sau mai nou).

Inainte de pornirea aplicatiei, este necesara rularea scripturilor sql, gasite in StoW/database, in ordinea in care sunt numerotate.

In plus, este necesara configurarea serverului local pentru a putea rula PDO impreuna cu Oracle.
